Inserindo vídeos hospedados localmente

Este vídeo está hospedado no meu próprio servidor.

Semelhante ao audio, podemos usar a tag <video> para inserir vídeos no site. Também devemos utilizar a tag <source> para testar vídeos com formatos diferentes.

Também temos alguns atributos que podemos colocar em <video> como "width", que define um tamanho para o vídeo e "poster" para colocar thumbnails ao vídeo.

Inserindo vídeos do YouTube

Para inserir vídeos do YouTube basta , no vídeo escolhido, selecionar a opção de incorporar, copiar e colar o código HTML disponibilizado.
