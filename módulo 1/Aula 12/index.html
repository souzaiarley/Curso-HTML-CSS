Usando links externos

Você pode acessar os meus repositórios públicos no GitHub.

Você também pode acessar o meu perfil no Instagram.

Usamos a tag <a> para criar links. É recomendado que se o link for para um site que não é seu, utilize target="_blank" para que o link seja aberto em uma nova aba. Bem como rel="external".

Usando links internos

Esta é a primeira página do site. Se você quiser, pode acessar também a minha segunda página.

Você também pode acessar a nossa página de notícias.

Aqui também utilizamos target="_self", para que o link seja aberto na mesma aba.

Também utilizamos rel="next" para indicar que a página seguinte é a que está sendo linkada e rel="prev" para indicar que a página anterior é a que está sendo linkada.

Por último utilizamos rel="nofollow" para indicar que o link não é um link de confiança. Esta última tag foi usada para linkar ao Facebook na página 02.
