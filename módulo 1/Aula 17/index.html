Nessa terceira técnica, deixamos todas as configurações de estilo em um arquivo separado chamado "style.css". É nesse arquivo que colocamos os seletores das tags desejadas. Para utilizá-lo, basta criar um link no "head" para o arquivo css.

Note também que podemos misturar as três técnicas em um mesmo arquivo. Geralmente temos uma precedência de estilo na seguinte ordem: inline -> interno -> externo.
