Devemos utilizar a tag <li> para representar os itens da lista

Note que as listas podem ser ordenadas de diferentes métodos. Nos exemplos acima, usamos os tipos 1, A e I. Existem ainda dois tipos de ordernação que são: i e a, que são apenas variações dos tipos I e A em minúsculo. Para acessarmos esses métodos, devemos digitar type após a tag <ol> e atribuir o valor desejado.

Listas de Definição

Para representar listas de definição utilizamos a tag <dl>

Linguagem de marcação para a criação do conteúdo de um site.

Linguagem de programação para a criação de interações em um site.
